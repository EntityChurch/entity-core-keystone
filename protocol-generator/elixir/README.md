## What it is

A pure-Elixir peer: canonical-CBOR (ECF) codec, Ed25519/Ed448 identity + signatures,
capability authorization with chain attenuation, and the §4.1 handshake / §6.5 dispatch
machinery — built on OTP processes. **Zero runtime Hex dependencies:** crypto is OTP stdlib
`:crypto`; CBOR / Base58 / varint are hand-rolled; the peer layer is GenServer/process-based.
`mix deps.get` pulls nothing.

- **Codec:** hand-rolled canonical CBOR (`lib/entity_core/cbor.ex`) via binary
  pattern-matching (`<<major::3, info::5, rest::binary>>`) — float minimization (f16⊂f32⊂f64),
  length-then-lex map-key sort, recursive major-type-6 tag rejection. LEB128 + Base58
  hand-rolled.
- **Crypto — fully native, no FFI:** Ed25519, **Ed448**, SHA-256/384 all from OTP `:crypto`
  (OpenSSL). The crypto-agility **higher bar** (Ed448 `key_type 0x02`, SHA-384) is reached
  **natively from the default build** — the headline contrast with OCaml's hybrid-FFI Ed448
  (A-OC-002). RFC 8032 deterministic → cross-impl signature byte-equality.
- **Integers:** BEAM integers are arbitrary-precision, so the CBOR head-form carrier holds the
  full 0..2⁶⁴−1 range with **no native-int trap** (contrast OCaml int63 / C# ulong / TS bigint).
- **Concurrency (BEAM actor model):** a GenServer content-store/tree (serialized, atomic CAS),
  one process per connection (single writer per socket, §4.8 per-request dispatch, §6.11
  `request_id` demux + outbound reentry).

## Use

Two tiers (full surface in [`status/PHASE-S5.md`](status/PHASE-S5.md) §3):

```elixir
# Tier 1 — codec island: encode/verify ECF
alias EntityCore.{Cbor, Model, Hash, Identity}

entity = Model.make("system/peer", %{"name" => "alice"})
bytes  = Cbor.encode(Model.to_cbor(entity))          # canonical ECF bytes
hash   = Hash.content_hash(%{type: "system/peer", data: %{"name" => "alice"}})
id     = Identity.of_seed(seed)                       # Ed25519 identity + peer_id

# Tier 2 — full peer
alias EntityCore.{Peer, Transport}

peer            = Peer.create(seed, open_grants: false)
{:ok, lsock, _} = Transport.listen(7777)
Transport.accept_loop(peer, lsock)
```
