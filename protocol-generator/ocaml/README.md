## What it is

A pure-OCaml peer: canonical-CBOR (ECF) codec, Ed25519 identity + signatures, capability
authorization with chain attenuation, and the §4.1 handshake / §6.5 dispatch machinery —
no FFI, no libsodium, no C bindings. `opam install` pulls no system packages. (The
crypto-agility **higher bar** — Ed448 — is an *opt-in* sub-library that does link a C-ABI
`.so`; see below. The core peer is unaffected.)

- **Codec:** hand-rolled canonical CBOR (`src/cbor.ml`) — float minimization (f16⊂f32⊂f64),
  length-then-lex map-key sort, recursive major-type-6 tag rejection. LEB128 + Base58
  hand-rolled (no extra deps).
- **Crypto:** Ed25519 via `mirage-crypto-ec` (pure-OCaml-over-fiat-C); SHA-256/384 via
  `digestif`. RFC 8032 deterministic → cross-impl signature byte-equality.
- **Crypto-agility higher bar (Ed448):** the OCaml ecosystem has no conformant native Ed448
  (A-OC-002), so `key_type 0x02` is sourced from `libentitycore_codec` over the C-ABI v1.1 —
  a **hybrid** (Ed25519 + SHA-256/384 native, Ed448 FFI). It lives in the **opt-in
  `entitycore_agility` sub-library** (`src/agility/`, guarded by `EC_AGILITY=1`) so the core
  peer above stays FFI-free. Byte-verified 25/25 vs the agility corpus, against both the C and
  Rust FFI impls (`./run-agility.sh`; harness `test/agility.ml`).
- **Integers:** CBOR head-form carrier (`Uint`/`Nint of int64` read unsigned) — full
  0..2⁶⁴−1 range despite OCaml's 63-bit native int (A-OC-001).

## Use

Library module prefix is `Entitycore_codec`. Two tiers (full surface in
[`status/PHASE-S5.md`](status/PHASE-S5.md) §3):

```ocaml
(* Tier 1 — codec island: encode/verify ECF *)
open Entitycore_codec
let e      = Model.make ~typ:"system/peer" (Cbor.Map [ Cbor.Text "name", Cbor.Text "alice" ])
let bytes  = Cbor.encode (Model.to_cbor e)        (* canonical ECF bytes *)
let hash   = Hash.content_hash ~typ:"system/peer"
               ~data:(Cbor.Map [ Cbor.Text "name", Cbor.Text "alice" ]) ()
let id     = Identity.of_seed seed                (* Ed25519 identity + peer_id *)

(* Tier 2 — full peer *)
let peer        = Peer.create ~seed ~open_grants:false in
let sock, _port = Transport.listen ~port:7777 in
Transport.accept_loop peer sock
```
